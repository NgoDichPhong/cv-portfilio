JavaScript cơ bản trong 30 phút – từ biến, hàm tới DOM

Đăng ngày: 25/10/2025 · 6 phút đọc

JavaScript là ngôn ngữ biến website từ tĩnh thành động – nơi mọi thứ có thể phản hồi, chuyển động và tương tác. Trong bài này, mình sẽ giúp bạn nắm gọn các nền tảng cơ bản nhất chỉ trong 30 phút.

🔹 1. Biến (Variables) – nơi lưu dữ liệu

Biến là hộp để chứa thông tin. Bạn có thể dùng let, const hoặc var để tạo biến: 

let name = "Minh";
const age = 20;
var isStudent = true;

Quy tắc nhanh:

  • let – có thể thay đổi giá trị
  • const – không đổi (thường dùng nhiều nhất)
  • var – cũ rồi, ít dùng

🔹 2. Hàm (Functions) – khối lệnh tái sử dụng

Hàm giúp gom nhiều dòng lệnh lại thành một khối tiện gọi lại. Ví dụ: 

function sayHello(name) {
  console.log("Xin chào, " + name + "!");
}

sayHello("Minh");

Ngoài ra còn có cú pháp ngắn gọn hơn: arrow function 

const sum = (a, b) => a + b;

🔹 3. Làm quen DOM – cách JavaScript “chạm” vào HTML

DOM (Document Object Model) cho phép bạn truy cập và thay đổi nội dung của website: 

const title = document.querySelector("h1");
title.textContent = "Xin chào JavaScript!";
title.style.color = "orange";

👉 Bằng vài dòng code, bạn có thể làm website “phản ứng” khi người dùng nhấn nút, nhập dữ liệu hoặc rê chuột qua.

🔹 4. Ví dụ mini – nút đổi màu

<button id="btn">Đổi màu</button>

<script>
  const btn = document.getElementById("btn");
  btn.addEventListener("click", () => {
    document.body.style.backgroundColor = "lightblue";
  });
</script>

Và đó, chỉ với vài dòng đơn giản, bạn đã tạo ra tương tác đầu tiên bằng JavaScript!

🚀 Kết luận

JavaScript là nền tảng của web hiện đại – từ hiệu ứng đơn giản cho tới các ứng dụng lớn như Gmail hay Facebook. Chỉ cần hiểu biến, hàm và DOM, bạn đã đi được 50% chặng đường.
